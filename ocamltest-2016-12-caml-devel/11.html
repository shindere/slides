11. Environment modifiers

``principal'' is defined in ocamltest/builtin_modifiers.ml as follows:

let principal = [
  Append (flags, " -principal ");
  Add (compiler_directory_suffix, ".principal");
  Add (compiler_reference_suffix, ".principal"); ]

Two other modifiers (defined in the same file) are available:

  • expect: gives the right value to the script variable. The expect test is then built by (1) applying the expect modifier to the current environment and (2) running the script test.
  • testing: links with the testing library
